UNDER THE HOOD

To get our project up and running we combined a few key components such as...


  • MapQuest API: An API that we can use to access a map of the world; this enables us to link a JSON file which we use to populate the map with different sightings and data points. The data points act as markers to indicate where sightnings took place.
  • A geocode API: The Geocode API helps us identify sighting locations as accurately as possible; From country, to state, to city, all the way down to the exact coordinates.
  • A bootstrap plugin: Allows us to store the data submitted by other users to a local database, where they can log in to access the the web app and report sightings.
